# Challenge 1: Data Types

## 🎯 Goal
Understand how to store and update variables in Solidity.

## 🧠 Concepts
- uint
- string
- bool

## 🛠️ Task
Create a function `setValues` that:
- takes a number, string, and boolean
- updates the contract variables

## 💡 Hint
Remember to use `memory` for string parameters.
